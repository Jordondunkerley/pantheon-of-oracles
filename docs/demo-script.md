# Pantheon of Oracles Desktop Prototype – Demo Script

## 1. Open the app
Lead with the core promise:
- Personal astrology becomes a living oracle council.

## 2. Show onboarding
Walk through:
- account setup
- birth data
- provider setup
- awakening a pantheon

## 3. Show astrology profile
Explain that the system is driven by real chart structure:
- planets
- angles
- houses
- transits / motion / future dynamic state

## 4. Show oracle registry
Highlight that each oracle is not just a chatbot, but a profile with:
- astrology mapping
- archetype
- voice
- faction / role
- visual identity

## 5. Show session workspace
Use seeded Oryonos / Lunos conversations to demonstrate:
- different personalities
- different utility
- persistent session concept

## 6. Show provider model layer
Explain that the product is designed to be model-agnostic.

## 7. Show import pipeline
Demonstrate that real structured oracle data can be ingested rather than manually recreated.

## 8. Close on the roadmap
Emphasize that this desktop prototype is the foundation for:
- mobile app
- richer interaction
- multiplayer systems
- broader Pantheon of Oracles product expansion
