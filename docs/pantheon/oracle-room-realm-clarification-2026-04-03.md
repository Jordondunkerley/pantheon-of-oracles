# Oracle Room Realm Clarification

## Clarified product logic
Each oracle chamber is:
- a reflection of the oracle
- an extension of the oracle
- effectively the oracle’s personal realm

It does **not** shape, alter, or exert force on the oracle.

## Applied direction
- oracle room copy now frames entry as stepping into the oracle’s personal realm
- chamber/oracle language was corrected where it implied the room acted upon the oracle rather than expressed them

## Why it matters
This is foundational lore/product logic. The oracle room should feel like entering the oracle’s domain, not like the chamber is compressing or controlling the oracle.
