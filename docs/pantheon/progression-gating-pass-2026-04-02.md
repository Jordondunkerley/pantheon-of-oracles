# Progression Gating Pass

## Why this pass mattered
Scene-based layout alone is not enough. To feel more like a game, progression needs unlock logic and a sense that later scenes are earned rather than freely tabbed from the start.

## What changed
- added unlocked scene tracking to progression state
- disabled locked scene navigation
- made oracle selection unlock later chamber scenes
- introduced stronger next-step signaling through progression state

## Product effect
This gives the chamber a more game-like reveal structure and reduces the feeling that all content is just freely exposed admin UI.
