# Unlock Feedback Pass

## Why this pass mattered
A game-like progression model needs visible reward and unlock acknowledgment. Without that, state changes remain invisible and the chamber still feels too software-like.

## What changed
- added reward-moment cards to progression nav
- surfaced earned vs locked reward states
- made progression feel more acknowledged rather than silently updated under the hood

## Product effect
This gives the user more emotional feedback that something has been gained, crossed, or opened.
